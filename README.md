Я развернул простое Flask веб-приложение в облаке на платформе Render.com.

Что сделал:
1. Создал простое веб-приложение на Python с использованием Flask
2. Написал Dockerfile для контейнеризации приложения
3. Загрузил код на GitHub
4. Развернул приложение на Render.com, который автоматически собрал Docker образ
5. Приложение доступно по публичной ссылке

Используемые технологии: Python, Flask, Docker, Render.com
